# Payoo Banking App 📱

Payoo is a secure and user-friendly mobile banking application built with **Next.js**. It offers essential financial services like depositing funds, transferring money, and paying bills with a robust PIN-based security system.

## 🔗 Live Demo

Check out the live application here: **[https://payoo-app.vercel.app](https://payoo-app.vercel.app)**

## ✨ Features

- **Simple Login Interface**: Secure authentication using a predefined mobile number and password[cite: 1, 2].
- **Home Dashboard**: A clean overview of account balance and quick access to all banking services[cite: 5].
- **Add Money**: Seamlessly deposit funds into your PAYOO account from various banks[cite: 5].
- **Cash Out**: Withdraw money easily with mandatory PIN verification for safety[cite: 5].
- **Money Transfer**: Securely send money to other PAYOO users[cite: 5].
- **Bonus Rewards**: Track and earn promotional bonuses using coupon codes[cite: 5].
- **Transaction History**: A complete, real-time record of all your financial activities[cite: 5, 7].
- **PIN Security System**: Every transaction requires a 4-digit PIN authentication for enhanced security[cite: 1, 5].
- **Responsive UI**: Optimized for all devices using **DaisyUI** and **Tailwind CSS**[cite: 1, 4, 7].

## 🛠️ Tech Stack

- **Framework**: Next.js (React)[cite: 2, 3]
- **Styling**: Tailwind CSS & DaisyUI[cite: 1, 4]
- **Notifications**: React Toastify[cite: 6]
- **State Management**: React Context API[cite: 6]
- **Icons**: Font Awesome[cite: 1, 6]
